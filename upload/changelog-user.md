# Дорогие пользователи! 🎉

Мы с радостью сообщаем о запуске нового раздела в нашем приложении — **Библиотека изображений**!

Здесь хранятся фотографии наших товаров, а также изображения с ними, которые вы сможете использовать для создания карточек на маркетплейсах, через компонент редактора - Изображение.

## 🔍 Как попасть на новую страницу
Найдите на главной странице в правом верхнем углу кнопку с иконкой с изображением нескольких папочек — при наведении на нее появится всплывающая подсказка **Библиотека изображений**. Просто нажмите на неё, и вы окажетесь в новом разделе!

**Что вас ждет в новом разделе:**
- Удобный поиск по названию файла
- Гибкие фильтры по различным параметрам
- Сортировка изображений в нужном вам порядке
- Детальный просмотр с полной информацией о файле
- Слайдер для быстрого переключения между изображениями, не выходя из режима просмотра

Мы продолжаем активно улучшать Библиотеку, чтобы работать с изображениями было максимально удобно и приятно. Заходите, тестируйте, делитесь впечатлениями! ✨
