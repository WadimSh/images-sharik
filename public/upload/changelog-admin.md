# Уважаемые пользователи! 🛠️

Мы значительно расширили функционал страницы **Библиотека изображений**. Теперь работа с изображениями стала еще удобнее и эффективнее!

## 🚀 Что нового?

### 🔍 Поиск, фильтрация и сортировка
- **Поиск по названию файла** — быстро находите нужные изображения
- **Гибкие фильтры** — ищите по тегам, автору и артикулам
- **Сортировка** — упорядочивайте изображения по дате загрузки, имени файла или размеру

### 🏷️ Управление тегами в модальном окне «Изображение подробно»
При клике на любую картинку открывается детальный просмотр, где теперь доступно:

- **Добавление тегов** — нажмите на иконку с изображением этикетки, в шапке модалки справа, чтобы открыть удобное меню:
  - Выбирайте теги из предустановленного списка
  - Создавайте свои собственные теги
- **Удаление тегов** — просто нажмите на крестик рядом с тегом (кроме системных тегов)
